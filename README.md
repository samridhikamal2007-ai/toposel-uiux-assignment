### Design Approach & System Write-up

AURA is a premium, clean-beauty e-commerce brand designed to connect organic botanical wellness with clinical skin science. The homepage layout focuses on clean editorial structure, visual balance, and high accessibility.

1. Typography & Visual Identity:
   - Headings: Playfair Display (Serif) – Exudes luxury, elegance, and premium branding.
   - Body & UI: Inter (Sans-Serif) – High readability, geometric, ensuring clean interactions at all screen sizes.
   - Spacing: Built on an 8px spacing grid system to ensure visual consistency across paddings and margins.

2. Color Palette & Accessibility:
   - Background: Warm Sand (#FDFBF7) – Soft off-white to reduce eye strain.
   - Primary Accent: Sage Green (#5F7D6D) – Represents botanical roots, wellness, and trust.
   - Secondary / Hovers: Rose Copper (#C88A75) – Represents skin glow and premium highlights.
   - Core Text: Charcoal (#2C2C2C) – High contrast readability without the starkness of pure black.

3. Responsive Grid Adaptations:
   - Header / Nav: Desktop aligns items horizontally. Mobile collapses categories into a drawer menu and compresses search to preserve screenspace.
   - Hero Banner: Shifts from a side-by-side split layout to a stacked layout centering textual weight.
   - Categories Grid: Desktop renders a 4-column row. Mobile reflows into a 2x2 grid to prevent layout crowding and maintain easy tap targets.
   - Products & Promos: Desktop features 3-column product cards and side-by-side promotional banner. Mobile reflows cards vertically with persistent checkout buttons and stacked promo blocks.
